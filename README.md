  <h2 align="center"><strong>Современные языки программирования</strong></h3>



### Требования к запуску

- [node: 24+](https://nodejs.org/en)

- [git](https://git-scm.com/)

### Установка

1. Загрузите локальную версию репозитория
   ```sh
   gh repo clone Jeleshkafff/modern-programming-languages
   ```
2. Перейдите в загруженный репозиторий
   ```sh
   cd modern-programming-languages
   ```      
3. Запустите код:
   ```sh
   node main.js
   ```
